## 3) 미해결 리스크 / 가정

### 3.1 미해결 리스크

1. SRS 원문 내 손절 시점 표현이 전략 간 상이하다(전략1: 15:05 종가, 전략2/3: 15:00 손절). 본 LLD는 각 FR 원문을 그대로 분리 적용했으며, 구현 단계에서 공통화가 필요하면 ILD에서 명시 합의가 필요하다.
2. 거래일 캔들에 09:05/15:00/15:05 필수 시점이 누락된 경우 전략 판단 불가능성이 존재한다.
3. 전략3의 RSI 누락 구간이 길어질 경우 매수 기회 과소평가가 발생할 수 있다.

### 3.2 가정

1. 입력 캔들은 KST(Asia/Seoul) 기준, 5분 경계, 09:00~15:30 거래시간 필터를 이미 만족한다.
2. RSI 값은 MARKETDATA 모듈에서 14기간, 0~100 범위로 산출/정렬되어 전달된다.
3. 시드머니/수량 계산은 SimulationEngine에서 Decimal 정밀도로 수행된다.

## 4) 결론

TICKET-005의 전략 엔진 LLD 문서는 HLD 4.3/5.x/6.x/7.x/8.x 및 SRS FR-003~FR-006, NFR-008을 기준으로 작성 완료되었다. 티켓 수용 기준 1~6과 문서 작성 요구사항 1~10을 모두 충족하며, 미해결 리스크는 ILD/구현 단계에서 관리 가능한 수준으로 식별되었다.
